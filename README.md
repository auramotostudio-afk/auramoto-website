# AuraMoto Phase 1 Design System

Fresh Next.js 15 + TypeScript project following the uploaded `AGENTS.md` rules.

## Scope

Phase 1 only:

- Theme system
- Typography
- Design tokens
- Reusable UI components

Not included yet:

- Full pages
- Booking system
- Supabase
- Three.js scenes

## Run

```bash
npm install
npm run dev
```

```text
src/
  app/

```
